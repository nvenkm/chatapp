
# ChatAPP

A simple ChatApp assignment i created while learning socket.io.



## Installation

Install node_modules folder

```bash
  npm install
```
Install express and socket.io
```bash
  npm i express
  npm i socket.io
``` 
Run the Server
```bash
  node index.js
```
